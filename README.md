# Yarchat

Швидкий месенджер у дусі Telegram, оптимізований для роботи на повільному інтернеті (E/2G).

## Запуск локально
```
npm install
npm start
```

## Деплой на Render
1. Запуш цей код у свій GitHub репозиторій (заміни поточні файли).
2. На Render: Environment → додай змінну `DATABASE_URL` зі свого Neon.tech проекту (Connection string з паролем).
   Без цієї змінної чат працює, але історія повідомлень не зберігається між рестартами сервера.
3. Build command: `npm install`. Start command: `npm start`.

## Можливості
- Офлайн-черга повідомлень (відправляються автоматично при відновленні зв'язку)
- Стиснення зображень на клієнті перед відправкою (canvas, WebP/JPEG)
- Голосові повідомлення (Opus, дуже легкий формат)
- Service Worker — кеш інтерфейсу та медіа для роботи без інтернету
- Індикатор якості з'єднання
- Режим економії трафіку (сильніше стиснення фото)
- Реакції на повідомлення (подвійний тап)
- PWA — можна "встановити" на iPhone через Поділитись → На екран Домівка
